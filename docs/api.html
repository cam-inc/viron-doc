必須API

Vironサーバを開発するにあたって、まず3つのAPIを作成する必要があります。

  • 認証方式を取得する /viron_authtype
  • swaggerを取得する /swagger.json
  • グローバルメニューを取得する /viron

です。

次頁以降でそれぞれのAPIについて詳しく記述します。
